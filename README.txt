bit.ly Chrome Extension
=======================

This is the official bit.ly chrome extension.

Installing
==========

The current release can be installed from http://chrome.bit.ly/

Development releases can be build by getting the source and loading as an unpacked extension

Contributing
============

Short Description
===========
Shorten and Share, right from your browser, instantly


Features / Description
===========
-. Shorten and Share, right from the extension, instantly
-. Supports direct login to bit.ly
-. Supports sharing to all your linked accounts, such as Twitter and Facebook
-. Expands all bit.ly links, including white-label domains, such as 4sq.com
-. Works with HTTPS / SSL websites
-. Uses new bit.ly SSL oAuth for authentication. You're credentials are never sent as clear text.

Source Code: http://github.com/bitly/bitly_chrome_extension
